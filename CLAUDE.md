# saas-toolkit-mobile

Mobile development track extension for saas-toolkit. Part of the nCode course ecosystem.

## Stack

- **Framework:** React Native + Expo
- **Database:** Firebase (Firestore)
- **Auth:** Firebase Auth (Google + Apple sign-in)
- **Storage:** Firebase Storage
- **AI:** Firebase AI Logic (Gemini client-side)
- **Payments:** RevenueCat (in-app purchases + subscriptions)
- **Push Notifications:** Firebase Cloud Messaging (FCM) + Notifee
- **Styling:** NativeWind (Tailwind for React Native)
- **Builds:** EAS Build + EAS Submit

## Auth

Default auth providers are Google and Apple sign-in via Firebase Auth. Apple sign-in is required for iOS App Store submission if you offer any social login.

## Requires

Base plugin: `dangogit/saas-toolkit`
